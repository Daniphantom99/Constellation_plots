# Constellation Plots in KNIME: An Automated Scaffold-Based Workflow for Interactive Chemical Space Visualization
## Specifications: 
The workflow was implemented using KNIME Analytics Platform (v. 5.4.2), which can be downloaded for free at https://www.knime.com/downloads. Some nodes used are part of the KNIME Base Nodes, and some others are part of specific free-use extensions such as: *KNIME Statistics Nodes (Labs)*, *RDKit Nodes Feature* or *KNIME SBG Support*.
The workflow can be downloaded in - .knwf - format, which is an specific type file for KNIME. 
## Download and Import Guide.
Once the user has download the desire workflow, it can be import into KNIME Analytics Platform as follows:
  1. Open KNIME Analytics platform.
  2. Click on - *home* - button, next - *Local Space* -.
  3. Click on - *Import workflow* - button, search for .knwf file on downloads folder, click on - Open -.
  4. The workflow is ready to use!
Since the workflow requires specific nodes -not included in KNIME base-nodes - a *Missing extensions* pop-up appears, which require confirmation from user to automatically install the missing nodes.
